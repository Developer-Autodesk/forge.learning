# Файлы со стороны клиента (.NET Core)

Обычно приложение ASP.NET Core обрабатывает файлы из папки `/wwwroot/`, создайте ее, если еще этого не сделали. Давайте организуем эту папку вот так:

- `wwwroot/`: `.html`
- `wwwroot/js`: `.js`
- `wwwroot/css`: `.css`
